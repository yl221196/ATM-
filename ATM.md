# ATM项目

## 1. 开发设计

### 功能

1. 开户
2. 登录
   1. 存款
   2. 取款
   3. 转账
   4. 查询余额
3. 销户退出

## 2.实现规划

### 个人账户类

1. id号 bankcard
2. 密码 password
3. 姓名 name
4. 余额 balance

### 验证码类

​		随机产生一个六位字符串验证码并返回。

### ATM机类

#### 1. 开户

​		增加一个个人账户并初始化信息。

#### 2. 登录

​		输入id和密码进行判断，同时通过验证码辅助验证，如果验证码正确再进行id和密码判断。

1. 存款： 输入金额存入个人账户。
2. 取款： 从个人账户中取出对应金额，账户余额减少。 要判断取出的金额是否大于此时余额，大于则取款失败。
3. 转账： 输入接收方的id号，从己方账户中取款。
4. 查询：  
   - 显示本用户的id信息与姓名。
   - 显示账户余额。
5. 注销账户：退出登录的银行账户。
